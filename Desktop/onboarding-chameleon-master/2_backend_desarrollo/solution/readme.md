# Soluciones Oficiales

Esta carpeta contiene las soluciones oficiales para los ejercicios del bloque de Backend. Las soluciones serán subidas por el equipo docente al finalizar cada ejercicio o sprint.

---

## **Importante**

1. **No modifiques los archivos de esta carpeta:**  
   - Los tutores proporcionarán las soluciones una vez finalizados los ejercicios.

2. **Trabaja tus soluciones en la carpeta `exercise/`:**  
   - Cada ejercicio tiene su propia subcarpeta con un archivo `readme.md`.

---

## **Contenido**

```plaintext
solution/
├── 01-estructura-symfony/           # Solución oficial para el ejercicio 1
├── 02-dynamodb-doctrine/            # Solución oficial para el ejercicio 2
├── 03-api-platform-graphql/         # Solución oficial para el ejercicio 3
├── 04-resolvers-personalizados/     # Solución oficial para el ejercicio 4
└── 05-pruebas-bdd/                  # Solución oficial para el ejercicio 5
```

---

## **Liberación de Soluciones**

Las soluciones estarán disponibles después de que todos los alumnos completen los ejercicios.
