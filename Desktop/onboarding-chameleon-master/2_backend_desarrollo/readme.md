# Backend - Desarrollo

Este bloque se enfoca en la construcción del backend utilizando Symfony, Doctrine, y API Platform. Implementaremos una API GraphQL con pruebas basadas en TDD y BDD para validar la funcionalidad.

---

## **Ejercicios**

1. **Estructura inicial del proyecto Symfony**  
   Aprende a configurar y estructurar un proyecto Symfony con autenticación básica.

2. **Integración de DynamoDB con Doctrine**  
   Configura DynamoDB como base de datos mediante Doctrine y crea repositorios básicos.

3. **API Platform y GraphQL**  
   Configura API Platform para generar un esquema GraphQL y realiza pruebas básicas.

4. **Resolvers personalizados**  
   Implementa lógica personalizada en resolvers utilizando API Platform.

5. **Pruebas BDD con Behat**  
   Valida flujos clave con escenarios de pruebas BDD.

---

## **Cómo empezar**

1. Navega a la carpeta `exercise/` y selecciona el ejercicio correspondiente.
2. Sigue las instrucciones detalladas en el archivo `readme.md` de cada subcarpeta.
3. Genera los outputs esperados y compáralos con las soluciones oficiales.
