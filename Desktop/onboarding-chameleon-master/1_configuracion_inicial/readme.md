# Configuración Inicial

Esta categoría está diseñada para establecer un entorno básico para el desarrollo web, utilizando herramientas modernas como Docker y configurando servicios clave. Este bloque incluye ejercicios para instalar y configurar los componentes necesarios de manera controlada.

---

## **Ejercicios**

1. **Instalar Docker**  
   Aprende a instalar Docker y verificar su funcionamiento.

2. **Instalar PHP**  
   Configura PHP localmente como base para el desarrollo backend.

3. **Instalar Node.js con NVM**  
   Utiliza NVM para gestionar versiones de Node.js.

4. **Configurar Visual Studio Code**  
   Personaliza tu entorno de desarrollo con extensiones esenciales.

5. **Configurar Docker Compose**  
   Crea un entorno reproducible para backend y frontend utilizando Docker Compose.

---

## **Cómo empezar**

1. Navega a la carpeta `exercise/` y selecciona el ejercicio que deseas realizar.
2. Sigue las instrucciones detalladas en el archivo `readme.md` de cada subcarpeta.
3. Genera los outputs esperados y compáralos con las soluciones oficiales.

¡Buena suerte con los ejercicios!
