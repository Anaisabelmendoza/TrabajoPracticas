# Solución: Instalar PHP

Para verificar la instalación de PHP, ejecuta el comando:
```bash
php -v
```

**Salida esperada:**
```plaintext
PHP 8.4.0 (cli) (built: Dec 6 2024 10:00:00) ( NTS )
Copyright (c) The PHP Group
Zend Engine v4.4.0, Copyright (c) Zend Technologies
```

**Recursos Adicionales:**
- [Guía oficial de instalación de PHP](https://www.php.net/manual/en/install.php)
