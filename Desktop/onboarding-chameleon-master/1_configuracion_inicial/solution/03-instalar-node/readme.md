# Solución: Instalar Node.js

Para verificar la instalación de Node.js y npm, ejecuta los comandos:
```bash
node -v
npm -v
```

**Salida esperada:**
```plaintext
v22.1.0
8.12.1
```

**Recursos Adicionales:**
- [Guía oficial de instalación de Node.js](https://nodejs.org/)
- [Instalar Node.js con NVM](https://github.com/nvm-sh/nvm)
