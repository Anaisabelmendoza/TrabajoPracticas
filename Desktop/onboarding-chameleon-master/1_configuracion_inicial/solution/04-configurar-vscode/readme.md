# Solución: Configurar Visual Studio Code

**Plugins recomendados:**
1. Docker
2. Prettier - Code Formatter
3. PHP IntelliSense
4. Tailwind CSS IntelliSense
5. Apollo GraphQL

**Cómo comprobar la instalación de los plugins:**
- Ve a la pestaña de extensiones (Ctrl+Shift+X).
- Asegúrate de que los plugins mencionados están instalados y habilitados.

**Recursos Adicionales:**
- [Página oficial de Visual Studio Code](https://code.visualstudio.com/)
