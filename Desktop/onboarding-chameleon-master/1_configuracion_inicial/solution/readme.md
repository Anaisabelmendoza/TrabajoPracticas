# Soluciones Oficiales

Esta carpeta está reservada para las soluciones oficiales de los ejercicios del bloque de Configuración Inicial. Las soluciones serán subidas por el equipo docente al finalizar cada ejercicio o sprint.

---

## **Importante**

1. **No añadas ni modifiques archivos en esta carpeta:** 
   - Las soluciones aquí contenidas serán añadidas por los tutores.
   - Cualquier modificación en esta carpeta podría causar conflictos al actualizar el repositorio.

2. **Trabaja tus soluciones en la carpeta `exercise/`:**
   - Cada ejercicio tiene su propia subcarpeta en `exercise/`.
   - Sigue las instrucciones indicadas en los `README.md` correspondientes.

---

## **Contenido de esta Carpeta**

```plaintext
solution/
├── 01-instalar-docker/      # Solución oficial para el ejercicio 1
├── 02-instalar-php/         # Solución oficial para el ejercicio 2
├── 03-instalar-node/        # Solución oficial para el ejercicio 3
├── 04-configurar-vscode/    # Solución oficial para el ejercicio 4
└── 05-docker-compose-setup/ # Solución oficial para el ejercicio 5
```

---

## **Liberación de Soluciones**

- Las soluciones estarán disponibles en este directorio una vez que el ejercicio haya sido completado por todos los alumnos o equipos.
- Si tienes dudas sobre los ejercicios, consulta primero los recursos adicionales en el [README general](../README.md).

---

¡Recuerda que el aprendizaje está en el proceso de resolver los ejercicios, no en copiar las soluciones! 🎓
