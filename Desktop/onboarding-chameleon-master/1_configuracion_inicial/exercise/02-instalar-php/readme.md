# Ejercicio 2: Instalar PHP

En este ejercicio aprenderás a instalar y verificar PHP.

---

## **Objetivos**
1. Configurar PHP localmente.
2. Verificar que PHP está operativo.

---

## **Instrucciones**

### 1. Descargar e instalar PHP
Consulta las instrucciones para tu sistema operativo en la página oficial.

### 2. Verificar la instalación
Ejecuta `php -v` para confirmar que PHP está instalado correctamente.

---

## **Recursos Adicionales**
- [Documentación oficial de PHP](https://www.php.net/manual/en/install.php)
