# Ejercicio 1: Instalar Docker

En este ejercicio aprenderás a instalar y verificar Docker en tu equipo.

---

## **Objetivos**
1. Instalar Docker Desktop.
2. Ejecutar y verificar un contenedor.

---

## **Instrucciones**

### 1. Descargar e instalar Docker Desktop
Sigue las instrucciones en la página oficial de Docker para tu sistema operativo.

### 2. Verificar la instalación
Ejecuta `docker run hello-world` para asegurarte de que Docker está funcionando correctamente.
