# Ejercicio 3: Instalar Node.js con NVM

En este ejercicio aprenderás a instalar Node.js utilizando NVM (Node Version Manager) y verificar que está correctamente configurado.

---

## **Objetivos**
1. Instalar NVM para gestionar versiones de Node.js.
2. Configurar y verificar una instalación de Node.js.

---

## **Instrucciones**

### 1. Descargar e instalar NVM
Sigue las instrucciones en la página oficial de NVM para tu sistema operativo.

### 2. Instalar Node.js con NVM
Usa el comando `nvm install --lts` para instalar la versión LTS de Node.js.

### 3. Verificar la instalación
Ejecuta los siguientes comandos para confirmar que Node.js y npm están instalados correctamente:
```bash
node -v
npm -v
```
