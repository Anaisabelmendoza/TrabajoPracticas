# Ejercicio 4: Configurar Visual Studio Code

En este ejercicio aprenderás a instalar y configurar Visual Studio Code como tu entorno de desarrollo principal.

---

## **Objetivos**
1. Instalar Visual Studio Code en tu sistema.
2. Configurar extensiones esenciales para el desarrollo.

---

## **Instrucciones**

### 1. Descargar e instalar Visual Studio Code
Accede a la página oficial de Visual Studio Code y descarga la versión para tu sistema operativo. Sigue las instrucciones para completar la instalación.

### 2. Instalar extensiones
Abre Visual Studio Code y accede al Marketplace de extensiones. Busca e instala las siguientes extensiones:
- **Docker**: Para gestionar contenedores directamente desde VS Code.
- **PHP Intelephense**: Para soporte avanzado de PHP.
- **Prettier**: Para formatear código automáticamente.
- **ESLint**: Para analizar y corregir problemas de JavaScript y TypeScript.

### 3. Configurar el editor
Personaliza tu configuración accediendo a `Preferences > Settings` y ajustando opciones como:
- Auto guardado del código.
- Activación de linters.
- Integración con Docker.

---

- [Atom](https://atom.io/): Una opción gratuita y personalizable desarrollada por GitHub.
