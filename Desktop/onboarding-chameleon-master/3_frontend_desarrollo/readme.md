# Frontend - Desarrollo

Este bloque se enfoca en la construcción del frontend utilizando Next.js, TypeScript, y Tailwind CSS. Aprenderemos a conectar el frontend con el backend mediante GraphQL y a implementar pruebas unitarias y E2E.

---

## **Ejercicios**

1. **Inicializar proyecto con Next.js**  
   Aprende a configurar un proyecto Next.js con TypeScript y Tailwind CSS.

2. **Implementar diseño base**  
   Desarrolla el diseño principal de la aplicación con Tailwind CSS.

3. **Conectar con el backend**  
   Usa Apollo Client para integrar GraphQL y consumir datos del backend.

4. **Pruebas unitarias**  
   Implementa pruebas para validar los componentes principales con Jest.

5. **Pruebas E2E**  
   Realiza pruebas E2E con Cypress para validar flujos clave.

---

## **Cómo empezar**

1. Navega a la carpeta `exercise/` y selecciona el ejercicio correspondiente.
2. Sigue las instrucciones detalladas en el archivo `readme.md` de cada subcarpeta.
3. Genera los outputs esperados y compáralos con las soluciones oficiales.
