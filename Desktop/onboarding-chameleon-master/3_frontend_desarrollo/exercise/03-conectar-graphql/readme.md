# Ejercicio 3: Conectar con el backend

En este ejercicio conectarás el frontend con el backend utilizando Apollo Client y GraphQL.

---

## **Objetivos**

1. Instalar y configurar Apollo Client.
2. Consumir datos del backend mediante GraphQL.
3. Mostrar datos en un componente.

---

## **Instrucciones**

1. Instala Apollo Client:
```bash
npm install @apollo/client graphql
```

2. Configura Apollo Provider en `_app.tsx`:
   - Define el cliente GraphQL apuntando al backend.
   - Envuelve tu aplicación en el `ApolloProvider`.

3. Consume datos:
   - Crea una consulta GraphQL para obtener datos del backend.
   - Muestra los datos en un componente de Next.js.

---

## **Recursos Adicionales**
