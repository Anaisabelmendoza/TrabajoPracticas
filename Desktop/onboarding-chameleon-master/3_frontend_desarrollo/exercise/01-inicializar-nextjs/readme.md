# Ejercicio 1: Inicializar proyecto con Next.js

En este ejercicio aprenderás a crear y configurar un proyecto Next.js con TypeScript y Tailwind CSS.

---

## **Objetivos**

1. Crear un proyecto Next.js.
2. Configurar TypeScript.
3. Instalar y configurar Tailwind CSS.

---

## **Instrucciones**

1. Crea un nuevo proyecto Next.js con TypeScript:
```bash
npx create-next-app@latest frontend --typescript
```

2. Instala y configura Tailwind CSS:
   - Instala las dependencias necesarias:
   ```bash
   npm install -D tailwindcss postcss autoprefixer
   npx tailwindcss init -p
   ```
   - Configura `tailwind.config.js` para incluir rutas de archivos de Next.js.

3. Prueba la configuración:
   - Crea un componente simple y aplica clases de Tailwind CSS para validar que funciona.

---
