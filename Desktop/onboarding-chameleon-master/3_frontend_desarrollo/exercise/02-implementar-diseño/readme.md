# Ejercicio 2: Implementar diseño base

En este ejercicio diseñarás la estructura principal de la aplicación con Tailwind CSS.

---

## **Objetivos**

1. Crear componentes principales (header, footer, y barra lateral).
2. Aplicar un diseño responsivo usando Tailwind CSS.

---

## **Instrucciones**

1. Crea un componente para el **header** con enlaces de navegación.
2. Diseña una sección **hero** con texto y botones llamativos.
3. Añade un **footer** con enlaces de redes sociales.
4. Asegúrate de que el diseño sea responsivo para dispositivos móviles.

---

## **Recursos Adicionales**

- [Componentes de Tailwind CSS](https://tailwindui.com/components)
- [Guía de diseño responsivo](https://tailwindcss.com/docs/responsive-design)
