# Ejercicio 5: Pruebas E2E

En este ejercicio realizarás pruebas E2E con Cypress para validar flujos clave de la aplicación.

---

## **Objetivos**

1. Instalar y configurar Cypress.
2. Escribir pruebas E2E para validar la navegación y funcionalidad del frontend.

---

## **Instrucciones**

1. Instala Cypress:
```bash
npm install --dev cypress
```

2. Configura Cypress:
   - Ejecuta `npx cypress open` y añade un caso de prueba en `cypress/e2e`.

3. Escribe una prueba:
   - Verifica que la navegación entre páginas funciona correctamente.
   - Valida que los datos cargados desde el backend se muestran.

---
