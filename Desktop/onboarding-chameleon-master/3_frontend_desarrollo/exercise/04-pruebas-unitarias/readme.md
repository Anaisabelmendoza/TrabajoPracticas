# Ejercicio 4: Pruebas unitarias

En este ejercicio implementarás pruebas unitarias para validar componentes clave con Jest y React Testing Library.

---

## **Objetivos**

1. Instalar y configurar Jest y React Testing Library.
2. Escribir pruebas unitarias para componentes principales.

---

## **Instrucciones**

1. Instala las dependencias necesarias:
```bash
npm install --dev jest @testing-library/react @testing-library/jest-dom
```

2. Configura Jest:
   - Añade un archivo de configuración `jest.config.js`.

3. Escribe una prueba para el componente **header**:
   - Verifica que los enlaces de navegación aparecen correctamente.

---

## **Recursos Adicionales**

- [Jest](https://jestjs.io/)
- [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/)
