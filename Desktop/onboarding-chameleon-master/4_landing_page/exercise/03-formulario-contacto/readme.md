# Ejercicio 3: Formulario de contacto

En este ejercicio implementarás un formulario de contacto funcional que se conecte al backend.

---

## **Objetivos**

1. Crear un formulario de contacto en la landing page.
2. Conectar el formulario al backend mediante GraphQL.

---

## **Instrucciones**

1. Crea el formulario en el frontend:
   - Incluye campos para nombre, email y mensaje.
   - Usa validaciones en el frontend con `react-hook-form`.

2. Conecta el formulario al backend:
   - Define una mutación GraphQL para enviar los datos al servidor.
   - Usa Apollo Client para manejar la solicitud.

3. Muestra mensajes de éxito o error al enviar el formulario.

---

## **Recursos Adicionales**

- [react-hook-form](https://react-hook-form.com/)
- [Mutaciones en Apollo Client](https://www.apollographql.com/docs/react/data/mutations/)
