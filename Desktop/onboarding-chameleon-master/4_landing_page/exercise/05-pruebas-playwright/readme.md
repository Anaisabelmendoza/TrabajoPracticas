# Ejercicio 5: Pruebas funcionales con Playwright

En este ejercicio realizarás pruebas funcionales con Playwright para validar la funcionalidad de la landing page.

---

## **Objetivos**

1. Configurar Playwright en el proyecto.
2. Escribir y ejecutar pruebas funcionales.

---

## **Instrucciones**

1. Instala Playwright:
```bash
npm install --dev playwright
npx playwright install
```

2. Configura Playwright:
   - Genera un archivo de configuración con `npx playwright init`.
   - Configura navegadores y rutas de prueba.

3. Escribe y ejecuta pruebas:
   - Crea un archivo de prueba para validar la navegación entre secciones.
   - Verifica que el formulario de contacto funcione correctamente.
