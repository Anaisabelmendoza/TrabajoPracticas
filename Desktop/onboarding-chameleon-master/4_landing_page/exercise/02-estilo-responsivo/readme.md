# Ejercicio 2: Estilo responsivo con Tailwind CSS

En este ejercicio aplicarás un diseño responsivo a la landing page utilizando Tailwind CSS.

---

## **Objetivos**

1. Implementar un diseño profesional con Tailwind CSS.
2. Asegurar que la página sea responsiva en dispositivos móviles y tablets.

---

## **Instrucciones**

1. Aplica estilos a las secciones creadas en el ejercicio anterior:
   - Usa clases de diseño como `flex`, `grid`, y `space-y` para el layout.
   - Aplica clases de color, espaciado y tipografía para el diseño.

2. Implementa diseño responsivo:
   - Usa las clases `sm:`, `md:`, y `lg:` de Tailwind CSS para adaptar el diseño a diferentes dispositivos.

3. Valida el diseño:
   - Prueba la página en el navegador con herramientas de desarrollador para simular diferentes tamaños de pantalla.

---

## **Recursos Adicionales**

- [Guía de diseño responsivo en Tailwind](https://tailwindcss.com/docs/responsive-design)
