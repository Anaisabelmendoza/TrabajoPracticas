# Landing Page - Integración Final

En este bloque se desarrollará una landing page completa utilizando Next.js, Tailwind CSS, y GraphQL, con optimización de SEO y pruebas funcionales con Playwright.

---

## **Ejercicios**

1. **Estructura de la landing page**  
   Crea la estructura básica de la landing page con las secciones principales.

2. **Estilo responsivo con Tailwind CSS**  
   Aplica diseño responsivo a la página para adaptarse a diferentes dispositivos.

3. **Formulario de contacto**  
   Implementa un formulario de contacto conectado al backend.

4. **SEO y optimización de rendimiento**  
   Mejora el SEO y optimiza el rendimiento de la landing page.

5. **Pruebas funcionales con Playwright**  
   Realiza pruebas funcionales para validar la funcionalidad de la página.

---

## **Cómo empezar**

1. Navega a la carpeta `exercise/` y selecciona el ejercicio correspondiente.
2. Sigue las instrucciones detalladas en el archivo `readme.md` de cada subcarpeta.
3. Genera los outputs esperados y compáralos con las soluciones oficiales.
